What is an event? What is a ride? What is a trip? How can I create a trip to an event?
Events are the best way to organize many trips to one locations. For example, you would want to create an event for a sporting event. Community members can sign into rideshare and then offer and ask for rides to the sporting event.
To create a new event by clicking 'Create a new Event' in the top panel. You will be asked to select an event location and supply a few additional details. After your event is created, you can view it by visiting the Events page.
Rides are useful when you need to get from one location to another, but you need sometime to drive. For example, if you need to go from Decorah to Minneapolis next Saturday, then you should create a ride and see if anyone is willing to drive you.
It is also possible someone is offering a trip to where you want to go. Examine the map and the Rides page to find out.
You can create a ride by clicking 'Request a Ride'.
Create a trip if you have organized a ride from one location to another and have extra room. You can help the community and environment by offering trips. If anyone is interested in joining your trip, they will join as a passenger and you will be notified.
You can create a trip by clikcing 'Create a new Trip'.
You can create a ride to an event by finding the event on the map. Events are denoted by blue icons. Click on the icon and you will have the option to create a ride to that event.
